
This project demonstrates a simple LAN where 4 PCs are connected to a 2960 switch.

## Devices
- 1 Switch (2960)
- 4 PCs

## Objective
- Connect multiple PCs in a LAN
- Test communication using ping

## How to Use
1. Open the `.pkt` file
2. Assign IP addresses to PCs (same network)
3. Use Command Prompt → `ping` to test connectivity

## File
- `4pcs.pkt`
