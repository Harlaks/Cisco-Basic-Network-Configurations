# Home Network Setup

## Objective
To design and configure a basic home network in Cisco Packet Tracer for communication between multiple devices.

## Description
This lab demonstrates the setup of a simple home network using common networking devices such as a wireless router, PCs, laptops, and other end devices. The network is configured to allow communication between connected devices within the same home environment.

## Devices Used
- 1 Wireless Router / Home Gateway
- PCs / Laptops
- Smartphones / Tablets (if used)
- Printer / Other home devices (if used)
- Copper straight-through cables / Wireless connections

## Concepts Covered
- Home network setup
- Basic IP addressing
- Wired and wireless connectivity
- Device communication within the same network

## Network Details
- Network Address: `192.168.0.0/24`
- Default Gateway: `192.168.0.1`
- IP Assignment: Static / DHCP (depending on your lab setup)

## Steps Performed
1. Added a wireless router/home gateway in Packet Tracer.
2. Connected PCs and other devices using wired or wireless connections.
3. Configured the home gateway/router settings.
4. Assigned IP addresses manually or automatically using DHCP.
5. Verified communication between connected devices.
6. Tested network connectivity using `ping`.

## Verification
- Devices connected successfully to the home network
- IP addresses assigned correctly
- Successful ping between devices



## Result
All devices in the home network were able to connect and communicate successfully.

## Conclusion
This lab helps in understanding how a basic home network is designed and configured, including both wired and wireless device connectivity.
